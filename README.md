# CodeBurguer interface

Esse projeto é um E-Commerce de uma hambúrgueria que permite a criação de produtos, categorias, pedidos usuários... Ele possui uma API própria e dois banco de dados, um relacional (PostgreSQL) e outro não relacional (MongoDB), ambos com deploy na nuvem para que fique acessível para qualquer pessoa. Além disso possui uma interface feita em React e diversos react hooks.
Tecnologias utilizadas no projeto:
back-end: Node.js, Sequelize, MongoDB, PostgreSQL, Yup, JWT Token, Express, Rialway Eslint + Prettier
front-end: React.js, Axios, CSS, HTML, MUI, Vercel, styled-components, react-hook-form, react-router-dom, useState, useRef, useEffect, react-toastify, Eslint + Prettier
